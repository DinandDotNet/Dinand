Micro Portal project

One stop medical portal for a serie of programs to collect and analyse blood results.

Handsome

Control over the handle of the results from several brands of micro-labs send to the main system. The program checks the results after it has verified the state of the microlab. All results are stored in a central storage system. Logged-in user is the operator of the micro-lab. Crusial checking the results and the status of the results. Receiving back the aproval.


X-tracked

Several blood measurements providers like laboratory, micro analysis systems send there results to this system to store them in the main system. Overview and checksums will confirm the results are received and saved.

Debriever

All result will be analysed and judged. Add advise and warnings to the results.

pochi

a website who gives you the results of saved blood values depending on the purpose. New results are highlighted sinds last visit. Result can be exported in several formats to match the client care systems.

Z-Analyse

This system analyses the history of the data to create a logical timeline by showing the fluctuations. This can be on one file or on multiple files the produce information for imaging of medical staff, sales, used methodes etc.
